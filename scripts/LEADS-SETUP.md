# ADONE — заявки с формы в Google Таблицу + Telegram

Форма на сайте отправляет данные **автоматически** (клиенту не нужно жать Send в мессенджере).

## 1. Google Таблица

1. Открой [Google Sheets](https://sheets.google.com) → **Создать** таблицу (например «ADONE Leads»).
2. **Расширения** → **Apps Script**.
3. Удали код по умолчанию и вставь содержимое файла **`scripts/leads-webhook.gs`** из репозитория.
4. В редакторе выбери функцию **`setupSheet`** → **Выполнить** → разреши доступ к таблице.
5. Появится лист **Leads** с заголовками.

## 2. Telegram-бот (опционально, но удобно)

1. В Telegram напиши **@BotFather** → `/newbot` → получи **токен** бота.
2. Напиши своему боту любое сообщение.
3. Открой в браузере:  
   `https://api.telegram.org/bot<ТВОЙ_ТОКЕН>/getUpdates`  
   Найди `"chat":{"id":123456789}` — это **CHAT_ID** (для лички с ботом).
4. В Apps Script: **Проект** → **Настройки проекта** → **Свойства скрипта** → добавь:
   - `TG_BOT_TOKEN` = токен от BotFather
   - `TG_CHAT_ID` = твой chat id

Без этих свойств заявки всё равно попадут в таблицу, просто без пуша в TG.

## 3. Опубликовать webhook

1. Apps Script → **Развернуть** → **Новое развертывание**.
2. Тип: **Веб-приложение**.
3. **Выполнять от имени:** я  
4. **У кого есть доступ:** все  
5. **Развернуть** → скопируй **URL веб-приложения** (длинная ссылка на `script.google.com/.../exec`).

## 4. Вставить URL на сайт

В **`index.html`** найди строку:

```javascript
const LEADS_WEBHOOK_URL = '';
```

Вставь URL между кавычками:

```javascript
const LEADS_WEBHOOK_URL = 'https://script.google.com/macros/s/XXXX/exec';
```

Сохрани и залей обновлённый `index.html` на хостинг.

## 5. Проверка

1. Открой сайт → заполни форму → **Request consultation**.
2. Должно появиться: *Thank you! We received your request…*
3. В таблице — новая строка; в Telegram — сообщение (если бот настроен).

## Если не работает

- URL должен заканчиваться на **`/exec`**, не `/dev`.
- После правок в Apps Script сделай **Новое развертывание** (версия обновится).
- В браузере жёсткое обновление: Cmd+Shift+R.

## Безопасность

- **Не публикуй** токен бота в коде сайта — только в «Свойства скрипта» Apps Script.
- URL webhook в `index.html` виден в исходнике; для спама добавлена скрытая honeypot-поле.
